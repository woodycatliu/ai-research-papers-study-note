## 如何添加新論文

每篇論文包含以下檔案：
1. **英文筆記**: `論文名稱_en.md`
2. **中文筆記**: `論文名稱_zh.md` 
3. **中文導讀**: 存放於 `guides/` 資料夾
4. **讀書心得**: 可包含在導讀中或單獨建檔

請按照以下格式添加索引項目：
```markdown
- [論文標題](相對路徑/論文名稱_zh.md) - 簡短描述
  - 英文版本: [English](相對路徑/論文名稱_en.md)
  - 導讀: [中文導讀](../guides/年份/論文名稱_guide.md)
```
